# Sales-Performance-Dashboard
Developed an interactive Power BI dashboard to analyze loan business and credit performance across multiple states and branches.
The dashboard tracks key metrics such as logins, sanctions, disbursements, pending cases, rejections, and DPD status. Advanced slicers & Bookmark and drill-downs enable quick, data-driven decision-making. Built to support management with fast, accurate, and visually engaging analytics.


This Power BI dashboard delivers a comprehensive analysis of business, credit, and operational performance for a financial services organization, utilizing historical loan and transactional data to generate actionable insights across loan disbursement, credit case status, branch productivity, and regional performance.
